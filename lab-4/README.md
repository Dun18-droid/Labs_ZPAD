### Інструкція користувача
1. **Керування гармонікою:** Використовуйте повзунки `Amplitude`, `Frequency` та `Phase` для зміни амплітуди, частоти та фазового зсуву основної гармоніки відповідно.
2. **Керування шумом:** Повзунки `Noise Mean` та `Noise Covariance` дозволяють змінювати середнє значення та дисперсію накладеного шуму. Шум перераховується *тільки* тоді, коли ви змінюєте ці два параметри.
3. **Фільтрація:** Повзунок `Cutoff Frequency` керує частотою зрізу фільтра низьких частот. Зменшуйте це значення, щоб сильніше згладити графік.
4. **Відображення шуму:** Чекбокс `Show Noise` дозволяє тимчасово приховати зашумлений сигнал, залишаючи лише ідеальну та відфільтровану криві.
5. **Скидання:** Кнопка `Reset` повертає всі параметри до їх початкових значень.
