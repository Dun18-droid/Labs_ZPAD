# NOAA VHI Аналіз (Лабораторна робота №5)

Веб-додаток на Streamlit для візуального аналізу індексів здоров'я рослинності (VHI) за даними супутників NOAA. Дозволяє фільтрувати дані по областях України, рокам та тижням, генерувати часові ряди та порівняльні гістограми.

## Структура
* `app.py` — основний код веб-додатка (UI та обробка).
* `downloader.py` — завантажує vhi дані для веб-додатку.
* `requirements.txt` — список бібліотек.
* `lab5.ipynb` — скрипт для первинного завантаження даних.
* `vhi_data/` — папка з датасетами (створюється автоматично).

## Інструкція з розгортання

**1. Налаштування середовища (venv)**

python -m venv venv
# Активація (Windows): venv\Scripts\activate
# Активація (macOS/Linux): source venv/bin/activate
2. Встановлення залежностей

pip install -r requirements.txt
3. Підготовка даних
Перед запуском виконайте скрипт завантаження doawloader.py(python doawloader.py)(чи відповідну комірку в Jupyter), щоб згенерувати папку vhi_data із .csv файлами.

4. Запуск сервера

Bash
streamlit run app.py
Додаток автоматично відкриється у браузері за адресою http://localhost:8501.
