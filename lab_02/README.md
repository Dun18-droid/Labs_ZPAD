# Лабораторна робота №2: Підготовчий етап Data Science

## Опис завдання
Робота складається з двох частин:
1. Автоматизоване завантаження та очищення метеорологічних даних (VHI-індекс) з сайту NOAA.
2. Аналіз та фільтрація великого датасету споживання електроенергії (Individual Household Electric Power Consumption) з використанням бібліотеки pandas.

## Інструкція із запуску
1. Відкрийте термінал у папці `lab_02`.
2. Встановіть залежності: `pip install -r requirements.txt`
3. Запустіть Jupyter Notebook: `jupyter notebook`
4. Виконайте комірки у файлах `Lab_2.ipynb` та `lab2_part2.ipynb` по черзі. Дані будуть завантажені автоматично.
