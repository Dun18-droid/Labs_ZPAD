# Лабораторна робота №3: Візуалізація даних

Цей репозиторій містить виконання лабораторної роботи №3 з візуалізації багатовимірних даних. 

Для аналізу було обрано датасет **Restaurant Consumer Data**, який відповідає всім вимогам (Multivariate, Categorical/Integer/Real атрибути, містить пропущені значення).

## Завдання, які було виконано:
1. Здійснено Data Cleaning (очищення даних від пропущених значень).
2. Побудовано графіки залежності одного integer/real атрибута від іншого.
3. Побудовано гістограму за одним з атрибутів із розподілом на 5 заданих діапазонів.
4. Створено складні графіки для ефективної візуалізації багатовимірних даних.
5. Загальна кількість побудованих графіків: 5-8, що відображають ключові інсайти датасету.

## Інструкція із запуску

Відкрийте термінал у папці з репозиторієм (переконайтеся, що ви у гілці `lab-3`).
Встановіть залежності: `pip install -r requirements.txt`
Запустіть Jupyter Notebook: `jupyter notebook`
Виконайте комірки у файлі `lab3.ipynb` по черзі. Дані будуть завантажені або прочитані автоматично згідно з кодом.
