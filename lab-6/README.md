# Лабораторна робота №6

## Вимоги до системи
- **ОС:** Linux (тестовано на Ubuntu 20.04/22.04)
- **Залізо:** Наявність підключеної веб-камери.
- **ПЗ:** CMake, GCC/G++, OpenCV.

## Інструкція зі збірки та запуску
1. Виконайте скрипт попереднього встановлення залежностей:
   `./preinstall.sh`
2. Зберіть проект:
   `./build.sh`
3. Запустіть програму:
   `./run.sh`

## Керування
- `1` - Звичайний режим
- `2` - Інверсія кольорів
- `3` - Розмиття (Gaussian Blur)
- `4` - Фільтр Canny (контури)
- `q` або `ESC` - Вихід
- Повзунок (Trackbar) керує яскравістю у звичайному режимі.
