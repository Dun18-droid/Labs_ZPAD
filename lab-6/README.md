Лабораторна робота №6: OpenCV Video Processor
Ця програма є інструментом для обробки відеопотоку з веб-камери в реальному часі. Вона побудована на базі бібліотеки OpenCV та має модульну архітектуру, де кожен компонент (камера, обробка, інтерфейс) винесений в окремий клас.

Системні вимоги
Операційна система: Linux (тестовано на Ubuntu 20.04/22.04).  
Обладнання: Веб-камера, підключена до системи.  

Залежності: CMake (>= 3.10), GCC/G++, бібліотека OpenCV.  
Покрокова інструкція зі збірки
Щоб запустити проект на "чистій" системі, виконайте наступні кроки в терміналі:

1. Перехід у директорію проекту
Відкрийте термінал та перейдіть у папку, куди ви зклонували репозиторій:

cd (ваша директорія)

2. Налаштування прав доступу
Зробіть усі скрипти автоматизації виконуваними:

chmod +x preinstall.sh build.sh run.sh
3. Попереднє встановлення (Preinstall)
Встановіть усі необхідні бібліотеки та інструменти збірки одним скриптом:

./preinstall.sh

Цей скрипт автоматично запустить apt install для OpenCV та CMake.  

4. Компіляція (Build)
Зберіть проект за допомогою CMake та Make:

./build.sh
Після успішного завершення в папці з'явиться директорія build/ із виконуваним файлом.  

5. Запуск (Run)
Запустіть програму:

./run.sh
Керування програмою
Програма підтримує інтерактивне перемикання режимів обробки за допомогою клавіатури та миші:

Гарячі клавіші:
1 - Normal: Звичайне відображення без фільтрів.


2 - Invert: Режим інверсії кольорів (негатив).  


3 - Blur: Розмиття за Гаусом (Gaussian Blur).  

4 - Canny: Виділення контурів алгоритмом Кенні.  

5 - Sobel: Фільтр Собеля (градієнтне виділення меж).

6 - Binarize: Бінаризація (чорно-біле зображення).

7 - Glitch: Ефект зміщення R/G/B каналів.


q або ESC - Вихід з програми.  

Інтерактивні елементи:

Trackbar (Brightness): Використовуйте слайдер у верхній частині вікна, щоб змінити яскравість зображення в реальному часі.  


FPS Counter: У верхньому лівому куті екрана відображається поточна швидкість кадрів за секунду.  

Структура проекту
Проект реалізовано згідно з принципами ООП:  


CameraProvider - робота з апаратним забезпеченням камери.  


FrameProcessor - логіка застосування фільтрів.  


KeyProcessor - обробка вводу користувача.  


Display - візуалізація та UI елементи.
